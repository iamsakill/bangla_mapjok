# bangla_mapjok

বাংলা মাপজোক একটি সহজ, দ্রুত এবং ব্যবহারবান্ধব অ্যাপ যা দৈনন্দিন জীবনে প্রয়োজনীয় সব ধরনের মাপ এক ক্লিকে রূপান্তর করতে সাহায্য করে। এতে রয়েছে দৈর্ঘ্য, ওজন, তাপমাত্রা, এলাকা, সময়, স্পিড, কারেন্সি সহ আরও অনেক ইউনিট কনভার্সন—সবকিছু বাংলা ভাষায়। ছাত্রছাত্রী, ইঞ্জিনিয়ার, ক্রেতা বা সাধারণ ব্যবহারকারী—সবার জন্যই এটি পারফেক্ট।
